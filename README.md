Welcome to the Churn Analysis Dashboard, a data-driven Power BI project developed during my internship at Codtech IT Solutions.
This dashboard provides deep insights into customer churn behavior, highlighting churn trends, demographics, service usage, revenue contribution, and predictive churn indicators.
